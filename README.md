# hs01-baremetal-firmware-no-pid
Кастомная прошивка для паяльника Fnirsi HS-01 на базе 32-битного RISC-V чипа CH571F с частотой ШИМ 250 Гц. Построена на основе кусочно-нелинейной функции с учётом ТКС жала и возможностью настройки агрессивности нагрева. Простота кода обеспечивает абсолютную стабильность системы.
